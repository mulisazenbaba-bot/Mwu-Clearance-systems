# MWU Clearance System

A comprehensive digital clearance management system built for Madda Walabu University.

## 📋 About This System

The MWU Clearance System is a digital platform designed to streamline the academic clearance process at Madda Walabu University. This system eliminates the traditional paper-based approach, providing a faster, more efficient, and transparent way for students to complete their clearance requirements.

### Key Features
- **Digital Clearance Requests**: Students can submit clearance requests online
- **Real-time Tracking**: Monitor the status of your clearance in real-time
- **Multi-Department Integration**: Seamless coordination between Library, Dormitory, and Academic departments
- **Automated Notifications**: Get notified when your clearance status changes
- **Digital Certificates**: Download your clearance certificate once approved
- **Officer Dashboard**: Department officers can efficiently manage and process clearances

### How It Works
1. **Student** submits a clearance request online
2. **Three departments** (Library, Dormitory, Academic) verify the student's records
3. **Officers** approve or reject based on their department's requirements
4. **System** automatically updates status and notifies the student
5. **Student** downloads the digital clearance certificate when all departments approve
